# Security policy

## Reporting a concern

Do not report credentials, tokens, private keys, certificates, private contact details, or restricted data in a public GitHub issue. Send a concise private disclosure to jishanislammaruf62@gmail.com and omit sensitive values until a safe exchange method is agreed.

## Intentionally excluded material

Datasets and source patches, Hugging Face caches, trained checkpoints, raw predictions and logits, authorization/run-state records, local certificate bundles, private submission contact files, and environment secrets are intentionally excluded from this repository.

## Scope

This repository is a research artifact. It provides no security, safety, performance, or clinical-deployment guarantee. It must not be treated as a validated clinical system or threshold.
